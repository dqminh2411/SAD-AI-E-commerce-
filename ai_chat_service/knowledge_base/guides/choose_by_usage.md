# Hướng dẫn chọn laptop theo nhu cầu

Mục tiêu của guide này là giúp chatbot tư vấn nhanh, đúng nhu cầu thay vì chỉ “chọn máy mạnh nhất”.

## 1) Trước khi tư vấn: 5 câu hỏi bắt buộc
1) Ngân sách dự kiến?
2) Bạn dùng chính để làm gì (văn phòng / học tập / lập trình / đồ hoạ / gaming)?
3) Có cần **mỏng nhẹ** và **pin lâu** không?
4) Màn hình cần kích thước bao nhiêu (13–14 / 15–16 inch)?
5) Bạn có cần nâng cấp RAM/SSD về sau không?

## 2) Văn phòng + di chuyển nhiều (mỏng nhẹ)
Ưu tiên:
- Trọng lượng ~ **1.2–1.6kg** (tuỳ model)
- CPU dòng tiết kiệm điện (U)
- RAM **16GB**
- SSD **512GB**
- Pin tốt, sạc USB‑C (nếu có)

Phù hợp cho:
- Word/Excel/PowerPoint
- Học online, họp video
- Lập trình cơ bản

## 3) Học tập + đa nhiệm vừa
Ưu tiên:
- 16GB RAM để mở nhiều tab + app
- SSD 512GB
- Màn hình đủ sáng, góc nhìn tốt

Gợi ý thêm:
- Nếu học ngành kỹ thuật (CAD/Matlab): cân nhắc GPU mạnh hơn.

## 4) Lập trình / DevOps
Ưu tiên:
- RAM **16GB** (hoặc 32GB nếu chạy VM/containers nhiều)
- CPU nhiều nhân/luồng hơn giúp build nhanh
- SSD nhanh, ưu tiên có khả năng nâng cấp

Chatbot nên hỏi:
- Bạn có chạy Docker/VMWare không?
- Bạn code mobile/game/AI hay web?

## 5) Thiết kế 2D / ảnh / UI
Ưu tiên:
- Màn hình màu đẹp (ưu tiên độ phủ màu cao)
- RAM 16GB
- SSD 512GB–1TB
- GPU tầm trung (tuỳ phần mềm)

## 6) Dựng video / 3D / đồ hoạ - kỹ thuật
Ưu tiên:
- CPU hiệu năng cao (dòng H)
- GPU rời (tuỳ khối lượng công việc)
- RAM **16–32GB** hoặc hơn (nếu dự án nặng)
- SSD **512GB** trở lên nếu lưu footage/dự án

Lưu ý:
- Cần tản nhiệt tốt, máy sẽ nặng và quạt ồn hơn.

## 7) Gaming + streaming
Ưu tiên:
- GPU rời
- Màn hình 120Hz+ (144/165Hz nếu chơi FPS)
- RAM 16GB (stream thường xuyên nên 32GB)
- Tản nhiệt tốt

Chatbot nên hỏi:
- Bạn chơi game gì? (FPS/AAA/eSports)
- Có stream/record không?
- Bạn ưu tiên FPS hay ưu tiên nhẹ/đẹp?

## 8) Checklist chốt đơn (tóm tắt)
- Nhu cầu → quyết định CPU/GPU
- Di chuyển → quyết định cân nặng/pin
- Dữ liệu → quyết định SSD
- Tuổi thọ 3–5 năm → ưu tiên 16GB+ và/hoặc nâng cấp được
